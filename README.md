# Esercizi Sistemi Operativi

Raccolta di esercizi svolti durante il corso di **Sistemi Operativi**, organizzati per argomento.

## Struttura del repository

| Cartella | Argomento |
| --- | --- |
| `es1 - introduzione a linux e c` | Introduzione a Linux e C |
| `es2 - processi unix` | Processi Unix (`fork`, `exec`, `wait`) |
| `es3 - segnali unix` | Segnali Unix |
| `es4 - file unix` | File Unix |
| `es5 - pipe` | Pipe |
| `es6 - file comandi` | Script shell (Bash) |
| `es8 - java threads` | Thread Java (es7 non presente) |

## Tecnologie utilizzate

- **C** – esercizi su processi, segnali, file e pipe in ambiente Unix
- **Bash** – script di shell
- **Java** – programmazione concorrente con thread
